# JH Games — Portfolio Website

Browser games collection portfolio for JH Games.

## Overview

Static portfolio site showcasing 6 browser games with links to live demos hosted on Cloudflare Pages.

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Google Fonts (Orbitron, Inter)
- Canvas-based animated particle background
- Hosted on Cloudflare Pages

## Games Featured

| Game | Description |
|------|-------------|
| Throne Guard | Medieval tower defense — 70 stages, PvP, endless mode |
| Realm of Shadows | Pixel action RPG — dungeons, bosses, 4 classes |
| HAVOC | Tactical FPS — 8 agents, 14 weapons, 3 modes |
| Nitro Surge | Sci-fi kart racing — circuit tracks, power-ups |
| Pixel Dash | Retro platformer — 54 levels, 9 worlds |
| Random Song Player | Music player — YouTube integration |

## Local Development

Open `index.html` in a browser. No build step required.

## Deployment

```bash
npx wrangler pages deploy . --project-name=jh-games
```
